Arti Warna Tabel

  • Hijau (Target Tercapai)

    Saldo Naik + Lot Bulan Depan NAIK.

  • Oranye (Profit < Target)

    Saldo Naik (Uang masuk) + Lot Bulan Depan TETAP.

  • Merah (Rugi)

    Saldo Turun. Dalam mode Flat, Lot tetap.

Bedah Logika Teknis

Mode Flat

Kalkulasi keuntungan biasa. Basis Resiko (Lot) SELALU SAMA dengan Modal Awal.

Risk Base = Start Capital

Mode Capped

Basis Resiko naik jika Profit >= Target. Jika Oranye, saldo naik tapi resiko lot tetap.

Update Risk if >= Target

ROI Global

Persentase (%) dihitung berdasarkan Modal Awal, bukan saldo berjalan.

ROI = Sum(PnL) / Input * 100
